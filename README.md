# ScanSense 

ScanSense is an automated network and web security scanning tool built using Java Spring Boot.

It integrates powerful tools like **Nmap** and **Gobuster** to perform deep analysis and generate structured reports.

---

 Features

  -Port scanning using Nmap  
  -Service & version detection  
  -OS detection  
  -Directory enumeration using Gobuster  
  -Risk analysis and vulnerability insights  
  -Automated report generation  

---

## Tech Stack

- Java
- Spring Boot
- Nmap
- Gobuster
- HTML/CSS

---

## How to Run

1. Install dependencies:
   - Nmap
   - Gobuster

2. Clone the repository: 
git clone https://github.com/SaakshiSingh41/ScanSense.git
cd ScanSense

4. Run this project
mvn spring-boot:run
Open browser:
http://localhost:8080
